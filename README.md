# ERN Template

Express/React/Node template for quickstart on front-end projects. Database open

| Command  | Description  |
| ------------ | ------------ |
| npm run server  | Initialize nodemon (:3000)  |
| npm run react  | Initialize webpack/babel  |




#### Includes:
- express
- nodemon
- react:
- react-dom


- @babel/core
- @babel/preset-env
- @babel/preset-react
- babel-loader
- webpack
- webpack-cli
